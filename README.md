# HR-Analytics
# 📊 HR Analytics Dashboard | Power BI

## 📌 Project Overview

The HR Analytics Dashboard is an interactive Power BI project developed to analyze employee attrition, workforce demographics, salary distribution, and job satisfaction. This dashboard helps HR teams identify the key reasons behind employee attrition and supports data-driven decision-making.

---

## 🎯 Objectives

- Analyze employee attrition
- Monitor workforce statistics
- Identify high attrition departments
- Analyze salary distribution
- Understand employee demographics
- Improve HR decision-making

---

## 🛠️ Tools Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV Dataset

---

## 📂 Dataset Information

The dataset contains employee information such as:

- Employee ID
- Age
- Gender
- Department
- Job Role
- Education
- Monthly Income
- Years at Company
- Job Satisfaction
- Attrition
- Business Travel
- Performance Rating
- Overtime

## 📊 Key Insights

- Total Employees: **1470**
- Total Attrition: **237**
- Attrition Rate: **16.1%**
- Employees aged **26–35** have the highest attrition.
- Laboratory Technicians and Sales Executives show higher attrition.
- Employees earning below **5K** experience higher attrition.
- Research & Development has the largest workforce.
